## Stack Tecnológica

A **Solasstec** é construída com uma arquitetura moderna e modular, priorizando **escalabilidade**.  

### **Backend**
- **NestJS** — Framework Node.js para aplicações escaláveis e estruturadas em módulos.
- **Prisma ORM** — Camada de abstração para o banco de dados, com tipagem estática e migrações automatizadas.
- **PostgreSQL** — Banco de dados relacional robusto, utilizado para persistência de dados transacionais.

### **Frontend (Descontinuado)**
- **Next.js** — Framework React para renderização híbrida (SSR/SSG) e rotas dinâmicas.
- **Tailwind CSS** — Framework utilitário para estilização rápida e responsiva.
